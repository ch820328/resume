# Interactive Resume Presentation

A modern, interactive web-based presentation built with HTML, CSS, and JavaScript. It features a slide-based layout, smooth animations, a laser pointer effect, and image zooming capabilities.

## Features

*   **Slide Navigation:** Smooth scrolling between slides using keyboard arrows (Up/Down/Left/Right), mouse wheel, or navigation dots.
*   **Laser Pointer:** A visual laser pointer effect that follows the mouse cursor.
*   **Image Zoom:** Click on images to smoothly zoom them to the center of the screen. Click again or click the backdrop to close.
*   **Speaker Notes:** Hidden speaker notes that are only visible when printing.
*   **Print to PDF:** Optimized print styles to export the presentation as a clean PDF document.

## Quick Start

1.  Open `index.html` in any modern web browser.
2.  Use the **Arrow Keys** or **Mouse Wheel** to navigate.
3.  Click the **Export PDF** button to save as a PDF.
